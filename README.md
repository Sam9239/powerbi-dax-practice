# PowerBI DAX Practice

## Overview
Practice repo for DAX time intelligence patterns and Power BI modeling.

## Structure
- `/pbix` (Power BI files)
- `/data` (sample data if needed)
- `/notes` (learning notes)
- `/screenshots` (visual evidence)

## How to use
1. Open the `.pbix` in Power BI Desktop.
2. Refresh data.
3. Review measures in the Measures table.
